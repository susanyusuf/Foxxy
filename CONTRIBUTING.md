# Contributing to Fox Compass

Thanks for your interest — contributions make this project stronger. This guide explains how to get started.

## Getting the project locally

1. Clone the repo:

```bash
git clone <your-fork-url>
cd maintainer-compass
```

2. Install dependencies and run locally:

```bash
npm install
npm run dev
```

3. Open `http://localhost:3000` and explore the app.

## How to pick an issue
- Browse the `issues/` folder to find curated tasks labeled `good first issue` or `documentation`.
- Comment "starting work" on the issue and assign yourself with a PR link when ready.

## Coding and PR process
- Create a branch: `git checkout -b feat/short-description`
- Make changes, run `npm run build` and ensure the app builds
- Open a PR against `main` with description of changes and testing steps

## Community guidelines
- Be respectful and patient
- Ask clarifying questions on the issue if unsure
