# Issue Labels for Fox Compass

- good first issue: For simple tasks suitable for new contributors. Suggested color: #ffb36b
- bug: For confirmed bugs. Suggested color: #ff4d4f
- feature: For new functionality. Suggested color: #ffd166
- documentation: For docs and guides. Suggested color: #c0c0c0
- help-wanted: For tasks that need community attention. Suggested color: #ff9f43
- backlog: For low-priority tasks. Suggested color: #7f7f7f
