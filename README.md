 # Fox Compass

 Fox Compass is a contributor-first toolkit to help open-source projects onboard contributors, triage issues, and plan releases with minimal friction.

 ## Quick start

 ```bash
 npm install
 npm run dev
 ```

 ## How to contribute
 - See `CONTRIBUTING.md` for setup and workflow
 - See `issues/` for curated issue ideas ready for contributors
 - Use the `.github/ISSUE_TEMPLATE` files when opening issues

 ## Submission
 - Use `SUBMISSION_TEMPLATE.md` to copy answers into the GrantFox project-onboarding form
